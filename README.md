# 🏳️ Flag Explorer

**Try it live: [Flag Explorer on GitHub Pages](https://demidchebakov.github.io/FlagComporator/)**

A lightweight web application built with Vanilla JavaScript, HTML, and CSS that allows users to compare flags side-by-side. The project also features built-in multilingual support.

## 🌟 Features

- **Side-by-Side Comparison:** Easily select and compare different flags.
- **Multilingual Support:** Localization (i18n) is supported via the `locales` directory.
- **Responsive Design:** Optimized for both desktop and mobile viewing.
- **No Build Steps:** Fully written in Vanilla JavaScript and standard web technologies—no complex bundlers or frameworks required.

## 🛠️ Tech Stack

- **HTML5:** Standard semantic markup.
- **CSS3:** Custom styling for layout and flag presentation.
- **JavaScript (Vanilla):** Core application logic, DOM manipulation, and dynamic localization handling.

## 📂 Project Structure

```text
FlagComporator/
├── css/             # Stylesheets for the application's layout and design
├── js/              # Vanilla JavaScript files for logic and interactivity
├── locales/         # Translation files for multi-language support
├── index.html       # The main entry point of the application
└── .DS_Store        # macOS system file (can be ignored)
```
